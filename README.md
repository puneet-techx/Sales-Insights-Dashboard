## Sales-Insights-Dashboard

## 📊 Project Overview
Developed an interactive sales analysis dashboard using Microsoft Excel to track and analyze business performance.

## 🛠 Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## 📈 Dashboard Highlights
- KPI Metrics: Total Sales, Orders, Quantity, Profit
- Category-wise sales analysis
- Quarterly sales trends
- Interactive filters (Region, Category, Year)
- Dynamic and user-friendly interface

## 🔍 Key Insights
- Identified top-performing categories
- Analyzed regional sales performance
- Evaluated profit trends for better decision making
